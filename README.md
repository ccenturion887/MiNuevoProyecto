Registro y logeo (El Restaurante)

1. Menú Principal: Elige la opción 1 (Restaurante).

2. Registro: Selecciona 1 para Registrar un Nuevo Restaurante.

3. Datos: Ingresa el nombre y la dirección del nuevo local (ej., El Sabor, Calle 1).

4. Carga Automática: El sistema te llevará al Menú del Restaurante logeado.

5. Añadir Catálogo: Elige la opción 3 (Cargar Nuevo Producto).

6. Producto: Ingresa el nombre, precio y categoría de un producto (ej., Refresco, 4000.00, Bebida).

7. Repetir: Puedes repetir la Opción 3 para añadir más productos si lo deseas.

8. Salir: Cierra la sesión del restaurante (4) para volver al Menú Principal.
---------------------------------------------------------------------------------------------------
---------------------------------------------------------------------------------------------------
Registro, logeo y hacer un pedido (El Cliente)

1. Menú Principal: Elige la opción 2 (Cliente).

2. Registro: Selecciona 1 para Registrarte (ej., Juan).

3. Acceso: Vuelve al menú de Cliente y selecciona 2 (Iniciar Sesión) con tu nombre.

4. Hacer Pedido: Selecciona 1 para hacer un pedido (Selecciona restaurante, y productos del menu).

5. Listo: El pedido se crea con estado "pendiente". Cierra la sesión del cliente.
---------------------------------------------------------------------------------------------------
---------------------------------------------------------------------------------------------------
Gestión y Actualización (El Restaurante)

1. Menú Principal: Elige 2 (Restaurante) e Inicia Sesión con el nombre del local (El Sabor).

2. Ver Pedidos: Selecciona 1 (Ver Pedidos Activos) para ver el pedido de Juan.

3. Actualizar Estado: Selecciona 2 (Actualizar Estado de un Pedido).

4. Selección: Ingresa el (1 hasta el numero de pedidos generados) del Pedido que quieres cambiar.

5. Cambio: Elige el nuevo estado, por ejemplo, en camino o entregado.
